Foco: Visualización Avanzada (Boxplots), Análisis Comparativo, Equidad Educativa, Power BI.

Habilidad Demostrada: Dominio en el diseño de visualizaciones estadísticas complejas que comunican distribución de datos, y capacidad para transformar datos demográficos en insights accionables para la orientación de acciones institucionales.

Análisis Macroeconómico: Tendencias de Exportaciones Nacionales (Tableau)

Foco: Limpieza de Datos (Power Query), Inteligencia de Mercado, Análisis de Series de Tiempo, Tableau.

Este proyecto utiliza datos oficiales de Aduana de Chile para analizar el comportamiento y las tendencias de las exportaciones chilenas. La fase inicial requirió limpiar y estructurar grandes sets de datos con Excel y Power Query (Lenguaje M) para estandarizar categorías y optimizar el set de datos para el análisis.

El dashboard interactivo en Tableau Public permite explorar los indicadores macroeconómicos clave por región, producto y tiempo. Demostrando la capacidad de transformar datos brutos de una fuente gubernamental en insights de inteligencia de mercado que identifican los sectores y destinos económicos más relevantes.

Habilidad Demostrada: Dominio en herramientas de Reporting Enterprise como Pentaho, creando reportes de alta calidad y listos para imprimir. Habilidad Demostrada: Manejo riguroso de datos sensibles y estadísticas oficiales. Aplicación de lógica DAX para calcular métricas de cambio porcentual y la capacidad de comunicar información compleja de salud pública de manera clara y jerárquica.
